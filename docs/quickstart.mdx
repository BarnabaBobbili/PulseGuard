## Prerequisites

Before starting, ensure you have the following installed on your system:
- [.NET 8.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- Visual Studio 2022 (with .NET desktop development workload) or your preferred IDE (e.g., Rider).

## Installation

1. **Clone the deployment repository**

```bash
git clone https://github.com/your-username/PulseGuard.git
cd PulseGuard
```

2. **Open the project**

Open the `PulseGuard.csproj` file in Visual Studio.

3. **Restore Dependencies**

Restore the required NuGet packages by building the project. Ensure you have network connectivity to download `LiveChartsCore.SkiaSharpView.Wpf`.

```bash
dotnet restore
```

## Running the Application

You can launch the dashboard by either clicking `Start` in Visual Studio or running the following command from the project directory:

```bash
dotnet run --project PulseGuard.csproj
```

Once started, the application will begin simulating patient ECG and SpO2 readings natively within the WPF window.

<Note>
PulseGuard generates local background telemetry to simulate real patient vitals. No external hardware or API setup is required.
</Note>
